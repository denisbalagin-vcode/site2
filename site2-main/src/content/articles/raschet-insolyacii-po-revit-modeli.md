Расчёт инсоляции вручную — одна из самых трудоёмких задач при проектировании.
Formit берёт исходную **Revit-модель** и автоматически рассчитывает инсоляцию
для всех светопроёмов, формируя готовый отчёт для экспертизы.

## Что нужно для начала

- Актуальная Revit-модель проекта
- Корректно заданные помещения и светопроёмы
- Учётная запись в веб-сервисе Formit

## Порядок работы

1. Загрузите модель в сервис.
2. Укажите параметры расчёта (расчётный период, нормативы).
3. Запустите расчёт — сервис обрабатывает до **1000 светопроёмов в минуту**.
4. Скачайте готовый отчёт.

> Освоение сервиса занимает около 40 минут — не нужно быть экспертом по
> светотехнике, чтобы получить корректный результат.

## Итог

Автоматизация расчёта инсоляции экономит десятки часов на каждом проекте и
снижает риск ошибок при подготовке документации для экспертизы.
